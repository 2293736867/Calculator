# Calculator
这个是仿照win10自带的计算器制作的简化版本.是用Qt做的,直接把整个表达式输入然后得出计算结果.


详细介绍:[CSDN博客](https://blog.csdn.net/qq_27525611/article/details/103060329).


**注意!!!由于包含完整的单独可运行的exe,需要使用git lfs clone**
